# ĐỀ CƯƠNG: MIXIGAMING Shop

## 1. Giới thiệu
MIXIGAMING Shop bán game cho phép người dùng tìm kiếm và mua online.

---

## 2. Chức năng

### Admin
- Quản lý user
- Quản lý thể loại game
- Quản lý game
- Quản lý đơn hàng
- Quản lý key game
- Thống kê

### User
- Đăng ký / đăng nhập
- Xem game
- Tìm kiếm
- Giỏ hàng
- Thanh toán
- Quản lý tài khoản
- Kho game đã mua

---

## 3. Công nghệ
- Laravel (Breeze)
- MySQL
- HTML/CSS (Tailwind)/JS
- PayOS API

---

## 4. Mở rộng
- Gợi ý game
- Wishlist
- Review
- Thanh toán online qua quét mã QR
