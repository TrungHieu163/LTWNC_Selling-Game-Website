# ĐỀ CƯƠNG: MIXIGAMING Shop

## 1. Giới thiệu
MIXIGAMING Shop bán game cho phép người dùng tìm kiếm, mua và tải game online.

---

## 2. Chức năng

### Admin
- Quản lý user
- Quản lý danh mục game
- Quản lý game
- Quản lý đơn hàng
- Thống kê

### User
- Đăng ký / đăng nhập
- Xem game
- Tìm kiếm
- Giỏ hàng
- Thanh toán
- Quản lý tài khoản

---

## 3. Công nghệ
- Laravel (Breeze)
- MySQL
- HTML/CSS/JS

---

## 4. Mở rộng
- Gợi ý game
- Wishlist
- Review
- Thanh toán online qua quét mã QR
